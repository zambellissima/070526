# Sistema de Rede de Cinemas

Projeto desenvolvido para a disciplina de Engenharia de Software.

## Objetivo

O sistema realiza o gerenciamento de cinemas, filmes e sessões de uma rede de cinemas.

## Tecnologias Utilizadas

- Python
- SQLite
- MVC
- Service Layer
- Repository Pattern

## Funcionalidades

- Cadastro de cinemas
- Cadastro de filmes
- Cadastro de sessões
- Registro de público
- Consulta de informações

## Estrutura do Projeto

/src
/docs

## Arquitetura

O sistema utiliza:

- MVC
- Service
- Repository

## Banco de Dados

SQLite
