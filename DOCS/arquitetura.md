# Arquitetura do Sistema

O sistema utilizará arquitetura em camadas baseada no padrão MVC.

# Estrutura

## View
Responsável pela interação com o usuário.

## Controller
Responsável por receber requisições da View.

## Service
Responsável pelas regras de negócio.

## Repository
Responsável pelo acesso ao banco de dados.

## Model
Responsável pelas entidades do sistema.

## SQLite
Responsável pela persistência dos dados.

---

# Fluxo da Aplicação

Usuário
↓
View
↓
Controller
↓
Service
↓
Repository
↓
SQLite
