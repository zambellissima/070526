# Casos de Uso

# Atores

## Funcionário / Administrador
Responsável pelo gerenciamento do sistema.

## Espectador
Responsável pela consulta de informações.

---

# Casos de Uso do Funcionário

- Cadastrar Cinema
- Consultar Cinema
- Cadastrar Filme
- Criar Sessão
- Registrar Público
- Consultar Público por Filme
- Consultar Público por Cinema

---

# Casos de Uso do Espectador

- Consultar Filmes
- Consultar Sessões
- Consultar Detalhes do Filme
